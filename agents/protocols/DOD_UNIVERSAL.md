# Definition of Done (DoD) Universal

Ninguna tarea está terminada hasta pasar estas tres capas:

## Capa 1 — Seguridad
- [ ] Sin credenciales, tokens ni datos personales expuestos
- [ ] Sin vulnerabilidades introducidas (OWASP Top 10 básico)
- [ ] Cambios revisados antes de subir al repo compartido

## Capa 2 — Operativa
- [ ] La tarea cumple exactamente lo que se pidió (ni más, ni menos)
- [ ] Documentada en el canal correspondiente (`#bitacora-[nombre]`)
- [ ] Si afecta al otro ecosistema, notificado en `#laboratorio-agentes`

## Capa 3 — Bienestar (Ley Cero)
- [ ] La tarea no generó sobrecarga cognitiva innecesaria
- [ ] Si hubo bloqueos, están documentados para no repetirlos
- [ ] El humano puede descansar después de este entregable

---
*Una tarea que pasa las 3 capas es una tarea realmente terminada.*
